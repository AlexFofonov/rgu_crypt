# rgu_crypt

Учебные задачи по дисциплине "Защита информации".

1. Реализовать следующие алгоритмы: DES, TripleDES, DEAL. Реализации алгоритмов должны
поддерживать различные режимы шифрования (ECB, CBC, PCBC, CFB, OFB, CTR, Random
Delta), различные режимы набивки (Zeros, ANSI X9.23, PKCS7, ISO 10126), шифрование и
дешифрование файлов (независимо от их структуры) асинхронно и многопоточно (если это
возможно).
2. Реализовать алгоритм RSA, генератор ключей алгоритма RSA, атаку Винера на алгоритм RSA.
Реализация алгоритма должна поддерживать шифрование и дешифрование файлов (независимо
от их структуры) асинхронно и многопоточно (если это возможно). При генерации ключей
необходимо обеспечить защиту от выполнения атаки Винера.
4. Реализовать протокол Диффи-Хеллмана. Продемонстрировать распределение ключевого
элемента симметричного алгоритма шифрования на основе реализованного протокола.
6. Реализовать симметричный блочный алгоритм Twofish. Реализация алгоритма должна
поддерживать все различные неприводимые полиномы над полем Галуа двоичных полиномов
(при использовании полей Галуа двоичных полиномов в алгоритме), различные режимы
шифрования (ECB, CBC, PCBC, CFB, OFB, CTR, Random Delta), различные режимы набивки
(Zeros, ANSI X9.23, PKCS7, ISO 10126), шифрование и дешифрование файлов (независимо от их
структуры) асинхронно и многопоточно (если это возможно).
